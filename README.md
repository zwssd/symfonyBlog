Blog Application
========================

个人开发的blog程序。

Requirements
------------

  * PHP 7.1.3 or higher;
  * PDO-SQLite PHP extension enabled;

```bash
$ cd symfonyBlog/
$ php bin/console server:run
```

Tests
-----

Execute this command to run tests:

```bash
$ cd symfonyBlog/
$ ./bin/phpunit
```
